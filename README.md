# page-macaujc-hub

这个仓库用于归档和发布多个独立的 HTML 页面。页面内容各自独立，不针对任何特定域名或网站。

## 项目简介

本仓库的主要目的是集中存放一批静态 HTML 页面，方便统一管理、版本追踪与访问。页面可能涵盖资讯、工具、展示等多种类型，彼此之间没有强制关联。

## 目录说明

```
├── index.html          # 入口页面（如有）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源文件（如 CSS、JS、图片等）
```

- `pages/`：每个 HTML 文件为一个独立页面，可直接通过浏览器打开。
- `assets/`：存放页面共用的样式、脚本或媒体资源，便于维护。

## 页面归档说明

- 所有页面均为纯前端 HTML 文件，不依赖后端服务。
- 页面之间无链接跳转要求，可独立运行。
- 归档内容可能随时间更新或移除，历史版本可通过 Git 提交记录查看。

## 维护说明

- 欢迎提交 Pull Request 添加新页面或修正现有页面。
- 请确保新增页面不与已有页面冲突，且文件命名清晰。
- 如涉及外部资源引用，建议使用相对路径或可靠的 CDN 链接。
- 本仓库不收录包含恶意代码、广告弹窗或违反平台政策的页面。

## 如何使用

1. 克隆或下载本仓库。
2. 使用浏览器直接打开 `pages/` 目录下的 HTML 文件。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
